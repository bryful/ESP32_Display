# ディスプレイ PIN

* GND          GND
* VCC          3.3V
* SCL (SCLK) → GPIO13
* SDA (MOSI）→ GPIO12
* RES (RST)  → GPIO11
* DC        → GPIO10
* CS        → GPIO9
* BLK       → GPIO8


lovyan03/LovyanGFX@^1.2.7 を使用

BLをGPI08に繋いでいるが、LovyanGFXだとBLの指定ができなかったので、classのコンストラクタに

```
pinMode(8, OUTPUT);
digitalWrite(8, HIGH);

```
を入れて無理やり使ってる。これが嫌なら3.3Vピンに指せばいい。ただし、バックライトの制御はできなくなる。
